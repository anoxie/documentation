# Les commandes CLI

### Création du projet
```bash
ng new <project-name>
```
### Lancement du projet
```bash
ng serve --open
```
le flag open permet d'ouvrir la page du projet une fois compilé dans le navigateur

### Création d'un nouveau component
```bash
ng generate component <nom-du-component>
```

### Création d'un service
```bash
ng generate service
