# Objects

Il est possible en javascript d'appeler la propriété d'un objet de 2 manières différentes, soit avec un point, soit comme un tableau, le notation en tableau permet d'utiliser 
```javascript
var obj ={
	    a: "hello world", // obj.a || obj["a"]
	    b: 42,
	    c: true
	};
```
